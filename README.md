# Dino

This is the first game I have ever made! THis is a simple 2D platformer game made in Godot.   
As all simple platformer games, it has basic movements, enemies, collectibles and a Win screen!

Status: Level 1 completed [working on other levels and an infinite version]

You can play it [here!](https://summerhigh18.itch.io/dino)

Note: No AI was used to make this game totally. Assets used to make this game are made by me and those taken from internet have been considerably edited for personal touch.
